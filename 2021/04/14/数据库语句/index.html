#数据库：数据库就是将大量数据保存起来，通过计算机加工而成的可以高效访问的数据集合

常见数据库：

1、Orache: 甲骨文公司

2、SQL Server: 微软公司

3、DB2: IBM公司

4、PostgreSQL 开源

5、MySQL 开源

6、Access 微软公司【古董】

注：数据库语句都有相通之处

#数据库结构
库： 就是一堆表组成的数据集合

表： 类似Ecel,由行和列组成的二维表

字段： 表中的列

记录： 表中的行

单元格： 行和列相交的地方

在数据库里有一个系统库：information_schema

在它里面有个存着所有库名的表:schemata

schemata_name库名

在他里面有个存着所有表名的表：tables

表名

table_schema 表属于的库
table_name 表名
column_name 字段名

#数据库语句：

select * from 表： 从表里查询所有内容（ * 表示所有）

where: 有条件的从表中选取数据

and 或 or: 条件有点多时候从表中选取数据

order by: 根据指定的结果集/指定的列进行排序

limit k.n: 从第k+1行起显示n条记录

union select: 将多个表拼在一起

select * from sanhao where id=1 and 1=2

NUll

select * from sanhao order by 1

select * from 表1 union select * from 表2 （只能选取同等数量的字段）
