Matlab学习笔记

数据类型

  • 无穷 —inf/Inf
  • 非数字 — NaN(not-a-number)
  • 圆周率 — pi
  • 字符串 —name=(‘China’)用单引号括起来
  • 字符串数组 —char(‘china’,’amera’,’xiaoming’)
  • 整型 —uint8(unsign无符号)、uint16、uint32、
          uint64、int8、int16、int32、int64
  • 查看最值 — intmin(‘int8’) intmax(‘int64’)
  • 浮点数 —double (matlab中定义的数默认为double类型）、single
  • 查看浮点精度 —eps(‘sigle’)-精度,—realmin(‘double’)-下限,—-realmax(‘single’)-上限

变量

  • 运算符： + - * / ^(
  • 变量赋值： x=12 x=3*x-12;(不回显可在后面加分号）
  • 变量名： 命名规则与编程语言类似，最长63个字符，不要使用内置变量（如：length、sum、end、pi、i、j、eps、sin、cos、size)
  • 查看内置变量： iskeyword

常用内置函数

  • 开方： sqrt(x)

  • 开n次方： nthroot(x,n)

  • 指数: exp(n)，e的n次方

  • 绝对值： abs(x)

  • 对数：log(x),log10(x),默认底数为e

  • 三角函数：

    • 以弧度为单位：sin(x),cos(x)，tan(x)…
    • 以度为单位：sind(x),cosd(x)…

  • 近似函数：

    • 就近取整(四舍五入）round(x)
    • 向零取整（直接取整）fix(x)
    • 向上取整ceil()
    • 向下取整floor()
    • 取余rem(a,b)

  • 清除内存变量：

    • clear x,y,z
    • clear （all) 清除所有变量
    • clc 清空页面，但不清除内存中的变量

  • 查看内存中的变量：who（查看所有变量名） whos(查看所有变量信息）

向量

  • 行向量： A=[10,2,3,5]或A=[10 2 3 5]

  • 列向量： A=[10;2;3;5]

  • 转置： B=A(:) 或 B=A’

  • 查看大小：size(A)—几行几列,length(A)—几列

  • 切片表达式：

    • A(1:6)—从第一个取到第六个元素（矩阵先列后行）

    • A(1：2：8）–从第1到第8间隔为2取元素

    • A(1:end) –取到最后

    • A([1 3 5])–取指定位置的数

  • 创建线性分布的向量（数列）：

    • 1：2：100 – 从1取到100，间隔为2
    • linspace(1,100,50) – 1到100等间隔取50个数
    • 表达式跨行（计算式换行）：…(三个点）
    • 标量与向量运算：使用常规运算符即可，如：A+5 – A向量每个元素+5
    • 向量与向量运算：运算符前带点（加号不用加点），比如+ .* ./ .^（两个向量相应的元素作运算，并不是点积）

矩阵

  • 矩阵定义： A=[10,2;3,5] 或A=[10 2;3 5]
  • 多重表达：A=[1:2:11;0:5:25; linspace(10,60,6);67 2 43 23 3 1] •
  • 零矩阵：zeros(4,3)
  • 矩阵：ones(4,3) –元素全为1
  • 单位矩阵：eye(5)–5维 数量矩阵n*eye()
  • 矩阵转置：A’
  • 矩阵的分号表达式：A(m:n,1:3）–选取第m行到n行，第1列到第3列–，A(:,1:end)–所有行1-3列–，A(2:5,1:end-1)
  • 矩阵的离散取值：A(1:3,[1 3])–选取1-3行中第1列和第3列–；A(A>0)
  • 矩阵元素的删除：A(:,2:4)=[] –把空矩阵赋值给对应元素
  • 矩阵拼接：C=[A B]–行方向拼接，需行数一样

常用矩阵内置函数

  • 创建对角阵： v=[2 4 7]、diag(v)
  • 反向获取对角元素：v=rand(10,5) diag(v，（１）)－－生成随机矩阵，去对角元素，默认第１列开始取
  • 获取尺寸：size(A)
  • 改变形状：reshape(A,5,2) －－元素个数要一致
  • 最值： ［ｄ，ｎ］＝min(A) ［ｄ，ｎ］＝max(A) －－取每列最值，去整个矩阵最值用嵌套ｍｉｎ（ｍｉｎ（））－－输出的ｄ为最小值，ｎ为最小值所在的行
  • 求和：sum(A) －－同上
  • 排序：sort(A) －－每列分别排序
  • 中值：median(A)
  • 均值：mean(A)
  • 标准差：std(A)
  • 点积：dot(A,B)
  • 随机数：rand－－产生一个随机数（范围１——１０）－－,rand(1,5)－－产生随机矩阵－－,rand(10)－－１０阶随机矩阵－－,randperm(10)－－产生１０个１０以内的随机数－－randperm(10,3)－－１０以内３个数－－
  • 标准正态分布的随机数：randn(5,3) －－均值是０，标准差是１
  • 特征值和特征向量：[v,d]=eig(A)－－输出ｖ为特征向量矩阵，ｄ为特征值矩阵－－

矩阵运算

  • 矩阵加减运算：+-
  • 矩阵点乘点除：.*.／－－对应元素相乘－－
  • 矩阵乘法：*
  • 矩阵求逆：inv(A)

脚本使用

  • 脚本定义：.m文件

  • 脚本代码编写：注释% 运行：在命令行输入脚本名

  • 脚本函数编写：function [(返回值)]=test(传入参数)，结束用end–脚本名称和函数名称一致

  • 脚本函数运行：test()

    示例：求解函数
    f(X)=(x^4*sqrt(3x+5))/(x^2+1)^2

    matalab代码

    function [f]=a(x)

    f=(x.^4.*sqrt(3.*x+5))./(x.^2+1).^2;

    end

    注：若输入x为向量或矩阵，运算符要用.点号运算–向量与向量按元素相乘除

函数

  • 函数定义：function…end
  • 函数返回值
  • 函数参数
  • 匿名函数：name=@(args)expr
  • 匿名函数例子：求解方程
    f1=@(x,y)2x^2-4x*y+y^2
    f1(2,3)
    传入向量需用点号运算
  • 绘制二维图例子：meshgrid,mesh
